# 任务清单（AI 可执行粒度）

本目录任务按 **`tasks_01.md` → `tasks_02.md` → …** 顺序执行；同一文件内建议自上而下。

## 约定

- 每个 **`.md` 最多 10 条**任务。
- 每条含：**编号、目标、产出路径（若适用）、验收标准**。
- 单条任务应在一个会话内可完成（或明确仅差人工运行 GUI验收）。

## 手工验收（MVP / tasks_07）

1. 可选：删除本机用户数据目录下的 `english_lean.db` 后首次启动，应自动导入样例词库。
2. `python -m english_lean`：队列非空时出现卡片与默写框；**默写错误**时「下一张」不可用。
3. **默写正确**后「下一张」可用；进入下一词后默写区重置且需再次答对才能继续。
4. 完成当日队列后出现「今日完成。」（高级）可用 `sqlite3` 查看 `progress.next_review_at` 是否更新。
